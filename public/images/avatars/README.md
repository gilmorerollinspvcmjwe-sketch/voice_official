# TTS 音色真人形象图片目录

此目录存放 AI 真人形象图片，用于 TTS 试听页面展示。

## 文件命名规范

```
{voiceId}-avatar.jpg

示例:
- xiaoya-avatar.jpg      (小雅)
- mingyu-avatar.jpg      (明宇)
- sakura-avatar.jpg      (Sakura)
- james-avatar.jpg       (James)
```

## 图片要求

- **尺寸**: 400x400px (正方形)
- **格式**: JPG 或 PNG
- **风格**: 逼真真人图片 (AI 生成或专业摄影)
- **裁剪**: 圆形裁剪样式
- **质量**: 高清晰度，适合网页展示

## 推荐生成工具

- Midjourney
- Stable Diffusion
- DALL-E
- 专业 AI 头像生成服务
