# TTS 应用场景图片目录

此目录存放应用场景展示图片。

## 文件命名规范

```
{scenario-id}.jpg

示例:
- customer-service.jpg   (客服场景)
- sales.jpg              (销售场景)
- podcast.jpg            (播客场景)
- audiobook.jpg          (有声书场景)
- education.jpg          (教育场景)
- game-npc.jpg           (游戏NPC场景)
```

## 图片要求

- **尺寸**: 800x600px 或更高
- **格式**: JPG 或 PNG
- **风格**: 专业场景照片或示意图
- **内容**: 与场景描述相符的视觉呈现

## 推荐场景图片内容

| 场景 | 图片内容建议 |
|------|------------|
| customer-service | 客服中心、电话咨询、在线客服界面 |
| sales | 销售外呼、商务洽谈、CRM系统界面 |
| podcast | 播客录音室、主持人、音频波形可视化 |
| audiobook | 读书场景、故事插图、有声书播放界面 |
| education | 教学课堂、在线教育平台、课程界面 |
| game-npc | 游戏角色、对话界面、虚拟世界场景 |

---

**注意**: 请确保所有图片具有合法的使用权限。
